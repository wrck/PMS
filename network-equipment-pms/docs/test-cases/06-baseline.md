# 计划基线模块 测试用例

> **模块**: 计划基线  |  **用例数**: 3  |  **更新日期**: 2026-07-24

> **前置条件**: 已登录（admin / admin123），存在已创建的项目

### TC-6.1 基线列表

| 路由 | 操作 | API | 请求体 | 预期 | 关注 |
|------|------|-----|--------|------|------|
| /baseline/list | 进入基线列表页 | GET /api/baseline/list | - | 基线列表正常加载，显示基线名称、版本、创建时间等信息 | 列表分页、筛选条件、字段展示完整性 |

### TC-6.2 创建基线

| 路由 | 操作 | API | 请求体 | 预期 | 关注 |
|------|------|-----|--------|------|------|
| /baseline/list | 点击"新建基线"，填写表单并提交 | POST /api/baseline | `{"projectId":1,"name":"基线V1.0","description":"初始基线"}` | 基线创建成功，列表刷新显示新基线 | 表单校验、创建后列表刷新、版本号生成 |

### TC-6.3 基线偏差分析

| 路由 | 操作 | API | 请求体 | 预期 | 关注 |
|------|------|-----|--------|------|------|
| /baseline/diff/{baselineId} | 点击基线"偏差分析"进入详情页 | GET /api/baseline/{id}/diff | - | 显示当前计划与基线的偏差数据（进度、工期、成本等差异） | 偏差数据计算、差异高亮显示、数据准确性 |
