# 工作流与审批模块 测试用例

> **模块**: 工作流与审批  |  **用例数**: 5  |  **更新日期**: 2026-07-24

> **前置条件**: 已登录（admin / admin123）

### TC-7.1 待办中心

| 路由 | 操作 | API | 请求体 | 预期 | 关注 |
|------|------|-----|--------|------|------|
| /workflow/todo | 进入待办中心页 | GET /api/workflow/todo/list | - | 待办任务列表正常加载，显示任务标题、发起人、时间等信息 | 列表分页、待办状态筛选、任务办理入口 |

### TC-7.2 统一审批中心

| 路由 | 操作 | API | 请求体 | 预期 | 关注 |
|------|------|-----|--------|------|------|
| /workflow/approval-center | 进入统一审批中心页 | GET /api/workflow/approval/list | - | 审批列表正常加载，支持按状态（待审/已审）筛选 | 列表加载、状态筛选、审批入口 |

### TC-7.3 审批详情

| 路由 | 操作 | API | 请求体 | 预期 | 关注 |
|------|------|-----|--------|------|------|
| /workflow/approval-detail/{id} | 点击审批项进入详情页 | GET /api/workflow/approval/{id} | - | 显示审批单详情、审批意见、流程图、审批历史 | 详情字段完整性、流程图渲染、审批操作（同意/驳回/转办） |

### TC-7.4 审批历史

| 路由 | 操作 | API | 请求体 | 预期 | 关注 |
|------|------|-----|--------|------|------|
| /workflow/approval-history/{recordId} | 查看某审批记录的历史 | GET /api/workflow/approval/{recordId}/history | - | 按时间倒序展示审批流转历史节点（审批人、意见、时间、结果） | 历史节点完整性、时间排序、审批意见展示 |

### TC-7.5 字段权限配置

| 路由 | 操作 | API | 请求体 | 预期 | 关注 |
|------|------|-----|--------|------|------|
| /workflow/field-perm | 进入字段权限配置页 | GET /api/workflow/field-perm/list | - | 字段权限列表正常加载，可配置字段读/写/隐藏权限 | 权限配置保存、字段映射、表单联动 |
