# Checklist

## Phase 1: 架构基座搭建

- [ ] 父 pom.xml 正确声明 Spring Boot 3.x parent，所有子模块可编译通过
- [ ] pms-common 模块提供统一 Result 响应体、全局异常处理器、公共工具类
- [ ] pms-admin 启动模块可独立启动，`mvn spring-boot:run` 在 30 秒内成功
- [ ] MyBatis-Plus 分页插件与自动填充字段配置正确
- [ ] Redis 连接正常，分布式会话可读写
- [ ] HikariCP 连接池配置合理，数据库连接正常

## 用户认证与权限管理

- [ ] 用户登录接口返回 JWT Token，后续请求携带 Token 可通过认证
- [ ] BCrypt 密码加密生效，明文密码不可逆

- [ ] 数据权限拦截器按公司/部门维度正确过滤数据
- [ ] 登出后 Token 失效，Redis 会话清除
